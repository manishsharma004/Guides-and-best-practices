# Things to check before using an api

### Limits
1. Per second.
2. Hourly limit.
3. Daily limits.
4. Monthly limits

### Endpoints
1. Total enpoints.
2. Availability.
3. Authentication.
4. Request and response.
5. Response codes.

### Clients
1. Languages supported.
2. Exceptions thrown.
3. Throttling support.
4. Multi-threaded?

### Tokens
1. Auth type
2. Passing in request
3. Refreshing tokens.
4. Expiry time.
